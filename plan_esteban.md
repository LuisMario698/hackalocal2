# Plan de Trabajo - Esteban
**Foco principal:** Formulario de Reportes, Sistema de Tareas Comunitarias y Gamificación de Acciones

## Responsabilidades Principales
1. **Flujo de Creación de Reportes**
   - Armar la interfaz visual completa para crear un "Nuevo Reporte".
   - Unir las piezas conectando la lógica del GPS (Mario) y enviando el paquete final usando la BD (Gordo).
2. **Ecosistema de Gamificación**
   - Tabular y diseñar la entrega de medallas/puntos y rangos avanzados ("Eco-Héroe", "Eco-Ciudadano") no solo por emitir un reporte, sino por _acciones_.
   - Asignar Eco-Puntos a los usuarios que completan tareas difíciles.
3. **Módulo de Tareas / Eventos de Limpieza**
   - Construir las vistas para "Voluntariado" o Limpieza colaborativa. Un reporte pasa a ser una "Tarea" en la cual las personas pueden unirse y marcarla como completada para ganar una medalla.

## Tareas (Checklist rápido)
- [ ] Maquetar pantalla de Crear Reporte.
- [ ] Integrar botón de tomar foto nativa usando cámara (`expo-camera`).
- [ ] Configurar el sistema de asignación y verificación de tareas de limpieza de la comunidad.
- [ ] Diseñar UI/UX de Medallas por "Limpiar zonas" y Acciones sociales en el perfil del usuario.
